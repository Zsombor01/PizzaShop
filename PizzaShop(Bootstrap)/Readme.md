This project main goal is to represent a normal shopping cart, but i thought i would combine this with some Bootstrap to represent my current knowledge from that too.

The main page builds up from an index.html, style.css.
The [about us] part of the page contains an aboutus.html, aboutus.css, and an aboutus.js

The hole package for the shopping cart is an ordernow.html, ordernow.css, ordernow.js and a pizzas.js. The pizzas.js is the file that contains the currently available pizzas in the "store". Here you can add or delete pizzas as u want, the ordernow.js will recognise that.

I tried to use minimal amount of css for this project, because my main goal was to use Bootstrap as much as i can.




**
Ennek a projektnek a fő célja, hogy egy normál bevásárlókosarat ábrázoljon, de úgy gondoltam, ezt kombinálom Bootstrappel, hogy abból is képviseljem a jelenlegi tudásomat.

A főoldal egy index.html, style.css fájlból épül fel.
Az oldal "rólunk" része egy aboutus.html, aboutus.css és egy aboutus.js fájlt tartalmaz.

Az oldal "bevásárlókocsi" része egy ordernow.html, ordernow.css, ordernow.js és egy pizzas.js. A pizzas.js az a fájl, amely az „áruházban” jelenleg elérhető pizzákat tartalmazza. Itt tetszés szerint hozzáadhat vagy törölhet pizzákat, az ordernow.js ezt felismeri.

Igyekeztem minimális mennyiségű css-t használni ehhez a projekthez, mert a fő célom az volt, hogy a lehető legtöbbet használjam a Bootstrapet.
